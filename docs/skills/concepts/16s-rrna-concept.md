# 16S rRNA 基因

## 定义

16S rRNA 基因是原核生物核糖体小亚基的组成部分，长度约 1500bp，包含保守区和可变区，广泛用于微生物分类和系统发育分析

## 解释

16S rRNA 基因测序是微生物组研究中最常用的方法之一。其原理是：

1. **保守区**: 用于设计通用引物
2. **可变区**: V1-V9 九个高变区，包含物种特异性序列
3. **系统发育**: 通过序列比对构建进化树

常用的可变区包括 V3-V4 和 V4 区。


## 示例

- V4 区扩增引物：515F (5'-GTGCCAGCMGCCGCGGTAA-3')
- V3-V4 区扩增引物：341F-805R
