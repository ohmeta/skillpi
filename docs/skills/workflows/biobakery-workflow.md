# bioBakery Workflow

Huttenhower 实验室开发的微生物群落分析工具集和教程平台，提供 MetaPhlAn、HUMAnN 等工具的集成环境

## 基本信息

- **输入格式**: FASTQ (宏基因组或宏转录组)
- **输出格式**: 物种丰度表、通路丰度表
- **难度**: intermediate

## 使用工具

- MetaPhlAn
- HUMAnN
- fastp

## 步骤

1. 1. 质量控制（fastp）
2. 2. 物种分类谱分析（MetaPhlAn）
3. 3. 功能谱分析（HUMAnN）
4. 4. 下游统计分析
